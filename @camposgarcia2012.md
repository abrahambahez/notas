El autor establece una distinción entre los tres conceptos que dan nombre al artículo. 

![[racialización#^1e2d50]]

![[racialismo]]

El racialismo está presente en las luchas anti-racistas, pues como ya se dijo, únicamente implica la aceptación de una *raza* como algo dado en el sentido común. Una lucha desde este enfoque es una lucha *desde la raza*, aunque no esté explícitamente declarada. El mismo caso aplica para políticas públicas *racializantes* que tienen el mismo punto de partida.

![[racismo#^2088e5]]

A partir de estas distinciones, el autor sostiene que es posible establecer marcos de acción, políticas y luchas anti-racistas más efectivas y significativas, así como sensibles al contexto donde efectivamente han tomado forma de manera particular y compleja. De igual modo, permite atender no sólo crítica sino creativamente cada elemento, no sólo para buscar eliminar sus efectos perniciosos, sino para imaginar futuros posibles de mundos pos-raciales.
