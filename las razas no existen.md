El concepto de [[raza]] fue desarrollado históricamente 

Aunque las razas no existen, el racismo sí. Esto se debe a diversas razones que siguen reproduciendo la diferencia jerárquica. Por ejemplo, ==hay un hábito visual racializado==, en el que se miran cuerpos de otros y se les asigna una categoría racial a partir de representaciones sociales. De este modo, en la práctica se forman estructuras diferenciales que atribuyen características esenciales a grupos de personas, no sólo físicas, sino intelectuales y morales, entre otras. Como consecuencia se justifican tratos excluyentes, opresiones y explotaciones
